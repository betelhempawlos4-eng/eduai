Neural Networks — Backpropagation Backpropagation uses the chain rule to compute gradients layer by layer, flowing backwards from the output to the input. Key steps: 1. Forward pass → compute predictions 2. Compute loss (e.g. cross-entropy, MSE) 3. Backward pass → propagate gradients using ∂L/∂w 4. Update weights: w = w - α · ∂L/∂w Key insight: each layer only needs to know the gradient flowing into it from the next layer.
